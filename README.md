dotfiles
========

my collection of configuration files and scripts

```sh
	git clone --recurse-submodules git@github.com:orlandofra/dotfiles.git
```

### the install script
This install script will install the dotfiles (it will clone the repo again in a temp
dir...)
```sh
	./dotfiles_install
```

You can execute the install script from my server
```sh
	curl -sSL https://orgiesataniche.com/dotfiles_install.sh | bash
```
